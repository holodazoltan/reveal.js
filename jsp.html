Architecture

Lifecycle

jspInit() - same as servlet init()

_jspService() - same as servlet service() no override

jspDestroy() - same as servlet destroy()

Scope

  • Request, Session, Application same as servlet
    •
  • Page - restrict lifetime of attributes to the same page where it was created, 'pageScope' implicit object
    •
remove attributes from every scope
pageContext.removeAttribute ("attrName")
 remove attributes from page scope
pageContext.removeAttribute("attrName", PAGE_SCOPE);

Implicit objects

  • request - HttpServletRequest object
  • response - HttpServletResponse object
  • out - PrintWriter object send to the client
  • session - HttpSession object
  • application - ServletContext object
  • config - ServletConfig object associated with the page
  • pageContext - define xml element's attribute
  • page - this
  • exception - access to exception data

Scriplet

java language statements, variable, method declarations or expressions

syntax
<% code fragment %>

Declaration

declares java variables or methods

syntax
<%! declaration; [ declaration; ]+ ... %>

Expression

a scripting language expression evaluated, converted to a String, inserted where the expression appears in the JSP file

syntax - no semicolon
<%= expression %>

Comment

marks text or statements that the JSP container should ignore

syntax
<%-- This is JSP comment --%>

Directive

provide directions and instructions to the container

syntax
<%@ directive attribute = "value" %>
  • page - instructions about current JSP page, import, buffer, error, ...
    •
  • import - include a file during the translation phase
    •
  • taglib - uses a set of custom tags
    •

Action

control the behavior of the servlet engine

syntax
  • jsp:include - includes a file at runtime
  • jsp:useBean - finds or instantiates a JavaBean
  • jsp:setProperty - sets the property of a JavaBean
  • jsp:getProperty - append the property of a JavaBean to out
  • jsp:forward - forwards the requester to a new page
  • jsp:element - define xml element
  • jsp:attribute - define xml element's attribute
  • jsp:body - define xml element's body
  • jsp:text - write template text

Standard Tag Library (JSTL)

collection of useful JSP tags, encapsulates the core functionality common to many JSP applications

Core Tags

collect most commonly used JSTL tags

syntax
<%@ taglib prefix = "c" uri = "http://java.sun.com/jsp/jstl/core" %>
  • <c:out> - write it to out
  • <c:set> - sets the result of an expression evaluation in a 'scope'
  • <c:remove> - removes a scoped variable
  • <c:if> - conditional
  • <c:import> - retrieves url
  • <c:forEach> - iteration
  • <c:redirect> - redirect to url

Formatting Tags

to format and display text, the date, the time, and numbers for internationalized web

syntax
<%@ taglib prefix = "fmt" uri = "http://java.sun.com/jsp/jstl/fmt" %>
  • <fmt:formatNumber> - format number
  • <fmt:parseNumber> - parse number from string
  • <fmt:timeZone> - specify time zone
  • <fmt:setLocale> - store locale
  • <fmt:requestEncoding> - encode message

SQL Tags

interacting with relational databases

syntax
<%@ taglib prefix = "sql" uri = "http://java.sun.com/jsp/jstl/sql" %>
  • <sql:setDataSource> - define datasource
  • <sql:query> - run a query
  • <sql:update> - alter
  • <sql:transaction> - transaction

XML Tags

creating and manipulating the XML documents

syntax
<%@ taglib prefix = "x" uri = "http://java.sun.com/jsp/jstl/xml" %>
  • <x:out> - write it to out
  • <x:parse> - parse string
  • <x:if> - conditional
  • <x:transform> - xslt

JSTL Functions

number of standard functions, string manipulation

syntax
<%@ taglib prefix = "fn" uri = "http://java.sun.com/jsp/jstl/functions" %>
  • <fn:contains> - text contains
  • <fn:lenght> - text length
  • <fn:join> - text join
  • <fn:trim> - text trim

Custom Tags

user-defined JSP language element, tag is converted to operations on an object called a tag handler

  • Eliminates the need of scriptlet tag
  • Separation of business logic from JSP
  • Re-usability

Tag Handlers API

Tag Handlers example

inheriting TagSupport, overriding doStartTag()


package com.javatpoint.sonoo;  

import java.util.Calendar;  
import javax.servlet.jsp.JspException;  
import javax.servlet.jsp.JspWriter;  
import javax.servlet.jsp.tagext.TagSupport;  

public class MyTagHandler extends TagSupport{  
  
public int doStartTag() throws JspException {  
    JspWriter out=pageContext.getOut();//returns the instance of JspWriter  
    try{  
     out.print(Calendar.getInstance().getTime());//printing date and time using JspWriter  
    }catch(Exception e){System.out.println(e);}  
    return SKIP_BODY;//will not evaluate the body content of the tag  
}  
}

Simple Tag Handlers

JSP 2.0 specification, extends SimpleTagSupport, override doTag()


package com.tutorialspoint;

import javax.servlet.jsp.tagext.*;
import javax.servlet.jsp.*;
import java.io.*;

public class HelloTag extends SimpleTagSupport {
   public void doTag() throws JspException, IOException {
      JspWriter out = getJspContext().getOut();
      out.println("Hello Custom Tag!");
   }
}

  • many implicit objects

  • similar than js, no typecasting, implicit type conversions, double and single quotes are equivalent
