Servlet Lifecycle

init() - initialized only once

service() - handle requests as new thread, HTTP request >> doXXX(), process request, write response

destroy() - terminated only once

State handling

HTTP stateless - always separate connection

  • Cookie - unique session ID assign to client
    •  
    Cookie: user=1fsdfsdfsd654f123sdfs
  • Hidden Form Fields - hidden HTML form field along with a unique session ID
    •
  • URL Rewriting - extra data on the end of each URL
    •  
    url?name1=value1&name2=value2

Scope

  • Request - live only inside service(), 'request' implicit object
    •
  • Session - client connection time, 'session' implicit object
    •
  • Application or context - available everywhere, one per web application per jvm, 'application' implicit object

Global State

Set data-state="something" on a slide and "something" will be added as a class to the document element when the slide is open. This lets you apply broader style changes, like switching the page background.

State Events

Additionally custom events can be triggered on a per slide basis by binding to the data-state name. 


Reveal.addEventListener( 'customevent', function() {
	console.log( '"customevent" has fired' );
} );
